# Security and repository data policy

This repository contains only generic Sudoku logic and synthetic examples.

Do not commit:

- personal, customer, employee, tenant, or account data;
- internal server, database, schema, object, host, path, endpoint, or infrastructure names;
- credentials, tokens, connection strings, private keys, or message payloads;
- proprietary SQL text, execution plans, logs, or operational metadata.

When reporting a suspected security issue, describe the category and affected file or line without reproducing secrets or sensitive values.
